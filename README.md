
# Intelligent Student Placement Prediction & Readiness System

## Overview
A professional Machine Learning system that predicts **placement probability**
and provides **skill-gap recommendations** with model explainability support.

## Key Features
- Ensemble ML model (Random Forest)
- Probability-based prediction (not binary)
- Skill-gap analysis for improvement guidance
- SHAP-ready explainability module
- Interactive Streamlit dashboard

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit, SHAP (optional)

## How to Run
```bash
pip install streamlit pandas scikit-learn
streamlit run app.py
```
